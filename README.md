The **Global Cyber Threat Monitor** is a centralized "Powerhouse" dashboard designed for cybersecurity professionals, researchers, and enthusiasts. It aggregates live threat maps, provides real-time telemetry simulations, and utilizes Google Gemini AI to offer actionable remediation strategies and threat analysis.

## üöÄ Key Features

### 1. Unified Threat Map Hub
Access the world's most critical "Live" Cyber Threat Maps in one place without navigating dozens of bookmarks.
*   **Sources:** Kaspersky, Fortinet, Check Point, NETSCOUT, Digital Attack Map, and 20+ more.
*   **Visuals:** High-fidelity animations and parallax backgrounds.

### 2. Real-Time Telemetry & Intelligence
A simulated operations center providing insights into:
*   **Live Threat Counters:** Malware, Phishing, DDoS, IoT Botnets.
*   **Vulnerability Tracking:** SQL Injection, XSS, RCE, and active CVE trends.
*   **Deep Dives:** Detailed research pages for every threat category with links to trusted sources (OWASP, NIST, MITRE).

### 3. AI Threat Analyst (Powered by Gemini)
An integrated Chat Interface that acts as your personal Security Operations Center (SOC) assistant.
*   **Capabilities:** Explain complex threat vectors, analyze trends, and define terminology.
*   **Context:** Pre-prompted with a "Paranoid but Professional" security persona.

### 4. Remediation Lookup Engine
A robust search tool for actionable defense strategies.
*   **Input:** Search for a vulnerability (e.g., "Log4j") or attack type.
*   **Output:** Structured JSON-based reports containing:
    *   Severity Ratings.
    *   Step-by-step Remediation Plans.
    *   Recommended Prevention Tools.
    *   Verified References (CVE Details, NVD).
